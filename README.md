# Calculadora de Préstamo Simple

Aplicación por consola desarrollada en Java que permite calcular valores básicos relacionados con un préstamo: interés simple, total a pagar, cuota mensual y comparación de cuotas por plazo.

---

## Funcionalidades

- **Interés simple:** calcula el interés generado dado un préstamo, una tasa y un número de meses.
- **Total a pagar:** calcula el valor total que debe pagar el usuario incluyendo préstamo e interés.
- **Cuota mensual:** calcula el valor aproximado de cada cuota mensual.
- **Comparación por plazo:** muestra las cuotas para 6, 12 y 24 meses.

## Instrucciones de ejecución

1. Se realiza la clonacion del repositorio:
2. git clone https://github.com/EmanuelTorres23/Calculadora-Prestamo-Simple.git
3. se abre el proyecto en IntelliJ IDEA.

3. y nos aseguramos de que la carpeta `src` esté marcada como Sources Root. Para esto haz clic derecho sobre `src` → **Mark Directory as** → **Sources Root**.

4. Ejecutamos el archivo `Main.java`.

5. Seleccionamos una opción del menú e ingresamos los datos solicitados.

## Flujo colaborativo

Este proyecto fue desarrollado siguiendo el flujo de trabajo colaborativo con Git y GitHub:

- Cada integrante trabajó en una rama independiente.
- Cada rama fue subida a GitHub con `git push`.
- Cada integrante creó un Pull Request hacia `main`.
- El dueño del repositorio revisó, aprobó y mergeó cada Pull Request.
- Las ramas remotas fueron eliminadas después del merge.
